A simple tool for recolouring curves in Maya 2025.

INSTALLATION:
- Save the folder this file is in to your maya/scripts directory.
- Drag and drop ColourPicker_INSTALL.py into your Maya viewport.
- A button should be created in your "Rigging" shelf.

Pressing the newly added button should open the tool.

HOW TO USE:
Select the curves in Maya that you want to recolour.
Use the drop down menu in the tool to select which colour you want the curves to be.
Press the "Colour Control" button to recolour all selected controls.
